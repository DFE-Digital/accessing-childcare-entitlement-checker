Information and guides for developers working on the Accessing Childcare Entitlement Checker.

In this section, you will find:

- Core Guides: Instructions for setting up your local development environment (using .NET 10 and devcontainers), our standards for code quality, trunk-based branching strategy, detailed release process, GitHub workflow naming conventions, and a guide for contributing to this documentation site.
- Security & Operations: Detailed workflows for running automated security scans using OWASP ZAP and how we manage dependency updates via Dependabot.
- Infrastructure & State: A detailed walkthrough of how we use Azure Bicep to bootstrap the Terraform state storage and the ephemeral lock/unlock mechanism used to keep the state secure, as well as a guide on the conditional aspects of our Terraform configuration.

Use the navigation menu to explore these guides in detail.
