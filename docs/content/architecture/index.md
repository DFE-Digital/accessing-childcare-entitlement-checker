High-level design and architectural principles for the Accessing Childcare Entitlement Checker.

In this section, you will find:

- Application Architecture: An overview of the system's internal structure, including the ASP.NET Core MVC web application and the shared Rules Engine.
- Deployment Architecture: Detailed diagrams and descriptions of our cloud infrastructure on Azure, including Front Door, App Service, and Terraform management.
- Network Architecture: Details of our secure networking topology, including virtual network layouts, subnets, service endpoints, private endpoints, and edge routing.
- Constraints & Principles: The foundational technical and design principles that guide the development of this service, ensuring alignment with GDS and DfE standards.
- Non-Functional Requirements: Specific operational targets for performance, availability, and security.
- Deployed Infrastructure: Automatically generated detailed information of our Terraform configuration and cloud resources.
- Security Architecture: The security controls, roles, threat modelling, attack vector analysis, and data classification designed to keep the service secure.
- Technical Review Checklist: A comprehensive review checklist mapping the project's governance, build, deployment, infrastructure, software engineering, and security compliance.
